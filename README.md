## 🛠️ O Que Eu Faço? (Minhas Habilidades)

Para quem está de fora, programação parece um monte de letras confusas. Mas aqui está como eu vejo o que estudo:

*   **🎨 Arquitetura de Telas (HTML5 & CSS3):** É como desenhar a planta e escolher a decoração de uma casa digital. Eu cuido de onde as coisas ficam e das cores que deixam tudo bonito.
*   **🧠 O Cérebro dos Apps (JavaScript):** Dou vida às páginas! É o que faz os botões funcionarem quando você clica e cria as regras dos jogos.
*   **🧩 Lógica Divertida (Scratch):** Minha base para montar blocos de comando e entender o pensamento lógico como se fosse um quebra-cabeça.
*   **🗄️ Meu Cofre de Criações (GitHub):** Onde guardo meus códigos com segurança e mostro minhas evoluções para o mundo.
